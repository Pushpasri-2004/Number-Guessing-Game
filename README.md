# Number-Guessing-Game
This is a simple number guessing game built using Python.

## HOW IT WORKS:
- The computer randomly selects a number between 1 and 10.
- The user gets 4 chances to guess the number.
- If the guess is correct, the user wins.
- If not, the correct number is revealed.
- Option to play again after each round.

## Languages and modules used:
- PYTHON
- RANDOM MODULE

## HOW TO RUN:
1. Clone the repository
2. Run `python main.py`
